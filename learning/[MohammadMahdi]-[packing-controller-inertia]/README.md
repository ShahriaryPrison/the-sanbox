# Packing Controller - Inertia + JSON API

## هدف پروژه
یادگیری ساخت Controller پیشرفته در Laravel برای یک workflow واقعی (بسته‌بندی سفارشات) با ترکیب **Inertia.js** (برای صفحه اصلی) و **JSON API** (برای عملیات frontend).

---

## چه چیزی یاد گرفتم؟

- ساخت Controller برای عملیات خاص بیزینس (Packing Process)
- ترکیب Inertia Rendering + JSON Response در یک Controller
- Route Model Binding
- Eager Loading روابط Eloquent
- استفاده از `match` expression برای منطق وضعیت
- جداسازی منطق به متدهای خصوصی (Clean Code)
- ساخت داده‌های مناسب برای frontend
